# MedStore Wholesale System

## Overview

MedStore Wholesale System is a Python application designed to manage medicine inventory, sales, and restocking operations for a wholesale pharmaceutical distributor. The system reads medicine records from a text file, displays available stock, processes customer purchases, manages supplier restocking, generates transaction invoices, and automatically updates inventory after every operation.

The application supports sales by tablet or strip, applies a 5% discount on purchases of two or more strips of the same medicine, and maintains accurate stock records through file-based storage. The project demonstrates the use of Python programming fundamentals, including file handling, functions, data structures, input validation, exception handling, and modular programming.

## Features

* Display available medicines and stock levels
* Sell medicines by tablet or strip
* Automatic 5% discount on bulk strip purchases
* Generate customer sales invoices
* Restock medicines from suppliers
* Generate supplier restocking invoices
* Real-time inventory updates
* File-based data persistence
* Input validation and error handling
* Modular and maintainable Python code structure
